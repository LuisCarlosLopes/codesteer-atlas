# Setup

Este guia descreve os passos para configurar o ambiente de desenvolvimento.

1. Instale as dependências com `uv sync`.
2. Rode o script de bootstrap (`./setup.sh`).
3. Execute a indexação inicial do workspace.

Para entender o contexto geral antes de começar, volte para a
[Visão Geral](../overview.md).

Se quiser entender por que algumas dependências foram descartadas, veja a
[seção de alternativas](../decisions.md#secao-inexistente) — link de exemplo
para um anchor que não corresponde a nenhuma seção indexada.
