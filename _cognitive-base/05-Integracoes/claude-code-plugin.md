# Claude Code plugin

Distribuição via `.claude-plugin/` marketplace — modo remoto `uvx`.

## Uso no Atlas

- `/plugin marketplace add LuisCarlosLopes/codesteer-atlas`
- `/plugin install codesteer-atlas`

## Contrato

- Mesmo servidor MCP que [[mcp-clients]]
- Indexação: `uvx ... atlas-index --workspace .`
